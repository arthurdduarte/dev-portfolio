# Hybrid Transaction System

Enterprise-style hybrid transaction processing system designed to simulate a real-world corporate architecture integrating modern backend services, legacy processing, relational databases, and containerized infrastructure.

---

# Technologies

- Golang
- COBOL
- SQL Server 2022
- Docker
- Linux Containers
- REST APIs
- gRPC (future implementation)

---

# Project Objectives

This project aims to simulate an enterprise environment combining:

- modern transactional services,
- legacy batch processing,
- relational databases,
- distributed systems concepts,
- containerized infrastructure,
- enterprise backend architecture.

The project is being developed as an engineering laboratory focused on backend systems, infrastructure, integration patterns, and enterprise software concepts.
